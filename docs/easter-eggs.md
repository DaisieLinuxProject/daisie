# Daisie Linux: Upcoming Easter Eggs & Features Roadmap

This document outlines the upcoming custom features and easter eggs planned for **Daisie Linux**, combining industrial CLI aesthetics with thematic anime and pop-culture references.

---

## 1. The Domain Expansion Protocol (`sudo expand-domain -y`)

* **Concept:** A system-wide state change inspired by *Jujutsu Kaisen*, allowing users to trigger a cursed "Domain" mode directly from the terminal with root privileges.
* **Execution & Effects:**
* **Terminal/Visual Glitch:** Running `sudo expand-domain -y` triggers a 10-second glitch/slash animation across the screen, mimicking a spatial distortion effect accompanied by digital static and audio cues.
* **Wallpaper & Logo Mutation:** The system wallpaper dynamically shifts, and the minimalist ASCII/character-based deer head logo (`|`) gets overlaid with Sukuna's iconic facial tattoo markings.


* **Reversal Command:**
* Running `sudo shrink-domain -y` restores the system back to its normal, stable state.



## 2. The Radio Demon Protocol (`sudo radio --on`)

* **Concept:** A tribute to Alastor from *Hazbin Hotel*, integrating a vintage radio broadcast theme into the OS environment.
* **Execution & Effects:**
* Running `sudo radio --on` simulates a classic radio frequency tuning parasite sound effect.
* The system's primary UI accent color and the deer head logo temporarily shift from the default purple to Alastor's signature threat-laden crimson red.

* **Reversal Command:**
* Running `sudo radio --off` restores the system back to its normal, stable state.
